# Notes d'implémentation

## Ce que fait ce backend
- Traduction NLP -> JSON gouverneur
- Clarification des demandes incomplètes
- Exécution d'un gouverneur minimal côté API
- Reformulation par audience
- Audit minimal dans Supabase

## Ce que tu peux brancher ensuite
- vraies fonctions SQL du gouverneur
- RAG sur les tables learning/canonical_guidance
- auth JWT
- rate limiting
- queue de jobs pour ingestion documentaire
- observabilité (Sentry, OpenTelemetry)
